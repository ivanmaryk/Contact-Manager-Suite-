Contact Manager Suite — 7 языков, управление контактами
Коллекция из семи реализаций менеджера контактов с богатым функционалом:

CRUD операции: создание, чтение, обновление, удаление контактов

Поля: имя, фамилия, телефон, email, адрес, группа (семья, работа, друзья), заметки, фото (путь/URL)

Поиск по всем полям

Группировка и фильтрация по группам

Экспорт/импорт в формате vCard (.vcf) и CSV

Сортировка по имени или дате создания

Хранение в JSON-файле / IndexedDB / SQLite

Валидация email и телефона

Аватарки (инициализация из первых букв имени)

Языки и технологии
Язык	Интерфейс	Хранение	Особенности
Python	Tkinter GUI	JSON	Валидация полей, экспорт vCard через vobject, цветовая индикация групп
JavaScript	Веб (HTML/CSS)	IndexedDB	Drag & drop для аватарок, мгновенный поиск, адаптивный дизайн
Go	Веб (HTML/template)	JSON (файл)	Быстрый поиск, автодополнение, REST API (опционально)
Rust	CLI + диалоги	JSON (серде)	Цветной вывод в таблицу, обработка ошибок, автозаполнение
Java	Swing GUI	JSON (Jackson)	Таблица JTable с сортировкой, поиск, иконки
C#	Windows Forms	JSON + двоичный файл	DataGridView, контекстное меню, drag & drop для фото
PHP	Веб + MySQL	MySQL / JSON	Полнотекстовый поиск, пагинация, экспорт в CSV
